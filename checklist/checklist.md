# Mac setup checklist

1. [ ] Finder turn on folder bread crumbs
  - `View` >> `Show Path bar`
2. [ ] Setup bluetooth connections
  - Headphones
  - Mouse
3. [] vs-code Keyboard mapping to switch terminals
4. setup `.zshrc`
  - see file in root of this repo
  - 
5. Setup home folder in Finder:
  - 1. CMD + SHIFT + H
  - 2. CMD+ArrowUp
  - 3. Drag to side bar
6. install jq json parser
  - `brew install jq`
7. Change TextEdit to no format
  - `CMD` + `,`
  - New Document >> select radio: `Plain text`
8. Set screen shots to specific folder, follow link:
  - [Screen shot to folder](../screen-shot-to-folder/screen-shot-to-folder.md)
